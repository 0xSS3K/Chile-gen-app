# ChileGen - Generador de Identidad Sintética Chilena

Aplicación web para generar identidades sintéticas chilenas con datos realistas, incluyendo RUT, nombres, direcciones, teléfonos y emails.

## Características

- Generación de identidades chilenas completas
- RUT válido con dígito verificador
- Nombres y apellidos chilenos realistas
- Direcciones de ciudades chilenas
- Teléfonos y emails generados
- Interfaz moderna y responsive
- Configuración de género y edad

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React (iconos)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Build

```bash
npm run build
npm start
```
